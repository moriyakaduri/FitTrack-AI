# מסמך דרישות מוצר (PRD) - FitTrack AI

## 1. מבוא ותיאור כללי

מערכת "FitTrack AI" היא אפליקציית דסקטופ המיועדת למתאמנים ואנשי מקצוע בתחום הכושר והתזונה. המערכת מאפשרת מעקב אחר מדדים, ניהול תפריטי תזונה ואימונים, ומספקת יועץ כושר ותזונה חכם מבוסס בינה מלאכותית (AI) המשתמש במידע אישי ובמאגרי ידע מקצועיים (RAG).

## 2. קהל יעד

* מתאמנים חובבים ומקצועיים המעוניינים לעקוב אחר התקדמותם.

* מאמני כושר ותזונאים המעוניינים לנהל את נתוני המטופלים שלהם.

## 3. דרישות פונקציונליות (מסכים ותכונות)

* **מסך התחברות (Login):** כניסה מאובטחת למערכת עם אימות מול מסד הנתונים.

* **מסך ראשי (Dashboard):** הצגת סיכום יומי של קלוריות, חלבונים, וגרף התקדמות משקל (באמצעות QtCharts).

* **מסך ניהול נתונים:** טבלאות להזנת ארוחות, אימונים ומדדי גוף, כולל אפשרויות חיפוש וסינון.

* **מסך צ'אט עם היועץ (AI Consultant):** ממשק שיחה עם סוכן ה-AI המקומי לקבלת המלצות תזונה ותוכניות אימונים מותאמות אישית.

## 4. ארכיטקטורה וטכנולוגיות

* **ממשק משתמש (Frontend):** Python בשילוב PySide6 (עיצוב בתבנית MVP).

* **שרת אחורי (Backend):** FastAPI הממומש בתבנית CQRS (הפרדה בין פקודות כתיבה לשליפת נתונים).

* **מסד נתונים:** SQL Server (בהמשך בענן Somee).

* **רכיב בינה מלאכותית:** Ollama הרץ בתוך Docker, בשילוב ארכיטקטורת RAG לשליפת מידע אמין.
